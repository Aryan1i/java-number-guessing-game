# 🎯 Number Guessing Game (Java)

This is a simple console-based Number Guessing Game developed using Java and Object-Oriented Programming (OOP) principles.

## 📌 How the Game Works

* One **Guesser** enters a secret number.
* Three **Players** try to predict that number.
* The **Umpire** collects all inputs and compares them to determine the winner.

## 🚀 Concepts Used

* Classes and Objects
* Method Calling
* Conditional Statements
* User Input using Scanner
* Basic Game Logic
* Resource Management (Single Scanner usage)

1. Compile the program:

   ```
   javac GuesserGame.java
   ```
2. Run the program:

   ```
   java GuesserGame
   ```
